# Getting Started



## Pre-requisites

This is what you need before cloning this repo

- npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
   ```sh
    git clone https://github.com/mamv2137/todo-list-tailwind.git
   ```
2. Change directory
    ```sh
    cd todo-list-tailwind
    ```
3. Install Dependencies

     ```sh
    yarn install
    ```
4. Run App

     ```sh
    yarn start
    ```
